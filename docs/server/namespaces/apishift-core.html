Classes

Authorizer An interface containing functions to manage and trigger authorization processes when a certain architectural configuration is requested.

Configurations Contains the core configuration to make the system run
DatabaseManager Interface which holds and manages PDO instances
DataManager Provides an interface to manage data entries and sources while hiding implementation details & automatic caching
DataModelManager Interface that helps manage the structures and types of items and relations available in the system
Item Interface to interact with database items/relations on the system
Process Provides an interface to compile and manage procedural connections in the system Procedural connection are explained in the ARCHITECTURE_AND_DESIGN.md file
SessionState Manages everything about session states & their data structure
Status Interface containing functions that provide a standard for notifying about internal & procedure states with data attached for the client to understand
Task Interface to manage collections of processes and call them when needed, whether for authorization, as a request, etc.
